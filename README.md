# flask_crud_sqlite3

##### Install and first use SQLite3 #####
pipenv install Flask-SQLAlchemy

1- Install sqlite3
    sudo apt install sqlite3

2- To execute sqlite
    sqlite3

3- To exit from sqlite terminal
    .exit

4- Create a database 
    sqlite3 mydatabase.db

5- Consult database 
    .databases

6- Create a table
Ex: create table products(id INT primary key );

7- List tables
    .tables
